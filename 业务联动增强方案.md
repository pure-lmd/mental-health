# 大学生健康管理系统 - 业务联动增强方案

## 一、核心问题

**当前问题**: 各模块(测试、活动、科普、AI咨询等)独立运行,缺乏业务关联,没有形成完整的健康服务闭环。

**解决思路**: 以**"用户心理健康档案"**为核心主线,让所有模块围绕档案产生数据联动。

---

## 二、业务架构设计

### 2.1 核心业务主线

```
用户注册 → 心理测评 → 生成健康档案 → 个性化推荐 → 参与活动/学习 → 复测评估 → 档案更新
   ↓          ↓            ↓              ↓              ↓            ↓          ↓
 基础信息   初始评估    健康等级+建议   科普/活动推荐   打卡学习    效果对比   成长轨迹
```

## 三、具体业务联动方案

### 3.1 联动一: 测试 → 档案 → 个性化推荐

**业务流程**:
1. 用户完成心理测试后,系统自动更新健康档案
2. 根据测试结果(健康/良好/建议沟通),系统推荐不同内容:
   - **建议沟通(高风险)**: 推荐心理疏导科普文章 + AI咨询 + 减压活动
   - **良好(中风险)**: 推荐心理健康知识 + 预防性活动
   - **健康(低风险)**: 推荐进阶心理知识 + 分享活动

// 新增推荐服务
@Service
public class RecommendService {
    
    /**
     * 根据测试结果生成个性化推荐
     */
    public RecommendVO generateRecommend(Long userId, Integer healthStatus) {
        RecommendVO recommend = new RecommendVO();
        
        switch(healthStatus) {
            case 2: // 建议沟通 - 高风险
                recommend.setKnowledgeList(knowledgeService.getRecommendByTag("心理疏导", 5));
                recommend.setActivityList(activityService.getRecommendByTag("减压放松", 3));
                recommend.setSuggestAiChat(true);
                recommend.setSuggestMessage("检测到您最近心理状态需要关注,建议与AI心理咨询师聊聊");
                break;
            case 1: // 良好 - 中风险
                recommend.setKnowledgeList(knowledgeService.getRecommendByTag("心理健康", 5));
                recommend.setActivityList(activityService.getRecommendByTag("心理预防", 3));
                recommend.setSuggestAiChat(false);
                break;
            case 0: // 健康 - 低风险
                recommend.setKnowledgeList(knowledgeService.getRecommendByTag("心理成长", 5));
                recommend.setActivityList(activityService.getRecommendByTag("心理分享", 3));
                recommend.setSuggestAiChat(false);
                break;
        }
        
        // 记录推荐
        saveRecommendRecord(userId, recommend);
        
        return recommend;
    }
}

### 3.3 联动三: AI咨询 → 档案 → 预警机制

**业务流程**:
1. AI咨询时分析用户情绪和关键词
2. 如果检测到高风险情绪(如抑郁、焦虑),触发预警
3. 预警信息记录到档案,并推荐专业帮助

**实现方案**:

```java
// 在 AiChatController 增加情绪分析
@PostMapping("/chat")
public Result chat(@RequestBody AiChatDTO dto) {
    Long userId = UserContext.getUserId();
    
    // 1. 调用AI获取回复
    String aiResponse = aiChatService.chat(dto.getMessage());
    
    // 2. 情绪分析(简单关键词匹配)
    EmotionAnalysisResult emotion = emotionService.analyze(dto.getMessage());
    
    // 3. 如果检测到高风险,触发预警
    if (emotion.isHighRisk()) {
        // 记录预警
        warningService.createWarning(userId, emotion.getRiskLevel(), emotion.getKeywords());
        
        // 推荐专业帮助
        aiResponse += "\n\n⚠️ 检测到您可能需要专业帮助,建议联系学校心理咨询中心。";
    }
    
    // 4. 保存对话记录
    aiChatService.saveChat(userId, dto.getMessage(), aiResponse);
    
    return Result.success(aiResponse);
}

// 情绪分析服务
@Service
public class EmotionService {
    
    // 高风险关键词库
    private static final List<String> HIGH_RISK_KEYWORDS = Arrays.asList(
        "自杀", "不想活", "绝望", "崩溃", "抑郁", "焦虑", "痛苦", "无助"
    );
    
    public EmotionAnalysisResult analyze(String message) {
        EmotionAnalysisResult result = new EmotionAnalysisResult();
        
        // 简单关键词匹配(可升级为AI情绪分析)
        List<String> matchedKeywords = HIGH_RISK_KEYWORDS.stream()
            .filter(message::contains)
            .collect(Collectors.toList());
        
        result.setHighRisk(!matchedKeywords.isEmpty());
        result.setRiskLevel(matchedKeywords.size() >= 2 ? "HIGH" : "MEDIUM");
        result.setKeywords(matchedKeywords);
        
        return result;
    }
}
```
